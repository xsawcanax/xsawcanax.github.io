Dockerfile Example

Przykładowy Dockerfile

# Etap budowy FROM golang:1.16 as builder # Ustawienie katalogu roboczego wewnątrz kontenera WORKDIR /app # Skopiuj pliki z kodem źródłowym COPY . . # Zbuduj aplikację RUN go build -o myapp # Etap produkcyjny FROM alpine:latest # Ustawienie katalogu roboczego wewnątrz kontenera WORKDIR /app # Skopiuj binarny plik z poprzedniego etapu COPY --from=builder /app/myapp . # Domyślne polecenie do uruchomienia aplikacji CMD ["./myapp"]

Wyjaśnienie kroków

Kod Dockerfile składa się z dwóch etapów budowy:

  1. Etap Budowy: 
                    FROM golang:1.16 as builder
                WORKDIR /app
                COPY . .
                RUN go build -o myapp
  2. Etap Produkcyjny: 
                    FROM alpine:latest
                WORKDIR /app
                COPY --from=builder /app/myapp .
                CMD ["./myapp"]

Ten Dockerfile umożliwia efektywne zbudowanie obrazu Dockerowego, który zawiera tylko niezbędne elementy do uruchomienia aplikacji, co przyczynia się do zmniejszenia rozmiaru obrazu i zwiększenia wydajności.
