Komenda - docker container

                                                                                                                                                                               
KomendaOpis
`docker container run -d {nazwa_obrazu}`Uruchamia kontener w tle (tryb detached).
`docker container run -it {nazwa_obrazu}`Uruchamia kontener w trybie interaktywnym i podłącza terminal.
`docker container run --rm {nazwa_obrazu}`Uruchamia kontener i automatycznie usuwa go po zakończeniu działania.
`docker container run -p {port_lokalny}:{port_kontenera} {nazwa_obrazu}`Mapowanie portu lokalnego na port w kontenerze.
`docker container run -v {ścieżka_lokalna}:{ścieżka_w_kontenerze} {nazwa_obrazu}`Montuje katalog lokalny w kontenerze.
`docker container run --name {nazwa_kontenera} {nazwa_obrazu}`Uruchamia kontener z określoną nazwą.
`docker container rm {id_kontenera}`Usuwa kontener o podanym ID.
`docker container inspect {id_kontenera}`Wyświetla szczegółowe informacje o kontenerze o podanym ID.
`docker container start {id_kontenera}`Wznowienie działania zatrzymanego kontenera o określonym identyfikatorze.
`docker container stop {id_kontenera}`Zatrzymuje działanie kontenera o określonym identyfikatorze.
`docker container logs {id_kontenera}`Wyświetla logi kontenera o podanym ID.
