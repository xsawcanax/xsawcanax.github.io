Komenda - docker image

 
Komenda Opis
docker image ls Wyświetla listę dostępnych obrazów na systemie.
docker image pull {nazwa_obrazu}:{tag} Pobiera obraz z rejestru Docker Hub.
docker image push {nazwa_obrazu}:{tag} Wgrywa obraz do rejestru Docker Hub.
docker image build -t {nazwa_obrazu}:{tag} . Buduje obraz z bieżącego katalogu.
docker image rm {nazwa_obrazu} Usuwa obraz o podanej nazwie.
docker image inspect {nazwa_obrazu} Wyświetla szczegółowe informacje o obrazie.
