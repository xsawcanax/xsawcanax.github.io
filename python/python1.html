Podstawy składni Pythona

Python jest językiem programowania ogólnego przeznaczenia, który cechuje się prostotą i czytelnością składni. Oto kilka podstawowych elementów składni Pythona:

Zmienne:

W Pythonie zmienne są deklarowane poprzez przypisanie wartości do nazwy. Na przykład: x = 10. Zmienne mogą przechowywać różne typy danych, takie jak:

    int: Reprezentuje liczby całkowite, np. 5, -3, 100. float: Reprezentuje liczby zmiennoprzecinkowe, np. 3.14, -0.001, 2.0. str: Reprezentuje tekst, np. "Hello, World!", 'Python', "123". list: Reprezentuje listę elementów, np. [1, 2, 3], ['apple', 'banana', 'orange']. tuple: Podobne do list, ale są niemodyfikowalne (immutable), np. (1, 2, 3), ('a', 'b', 'c'). dict: Reprezentuje słownik, czyli zbiór par klucz-wartość, np. {'name': 'John', 'age': 30}.
Operatory:
    Arytmetyczne:
      + (dodawanie) - (odejmowanie) * (mnożenie) / (dzielenie) % (modulo - reszta z dzielenia) ** (potęgowanie)
    Porównania:
      == (równa się) != (nie równa się) < (mniejsze niż) > (większe niż) <= (mniejsze lub równe) >= (większe lub równe)
    Logiczne:
      and (i logiczne) or (lub logiczne) not (negacja logiczna)
    Przypisania:
      = (przypisanie wartości) += (dodaj i przypisz) -= (odejmij i przypisz) *= (pomnóż i przypisz) /= (podziel i przypisz) %= (oblicz resztę z dzielenia i przypisz)
Print: Funkcja print() służy do wyświetlania danych na ekranie. Na przykład: print("Hello, World!"). Warunki: Warunki są wyrażane za pomocą instrukcji warunkowych if, elif, else. Na przykład: 

if x > 10:
    print("x jest większe niż 10")
elif x == 10:
    print("x równa się 10")
else:
    print("x jest mniejsze niż 10")
Pętle: Python obsługuje pętle for i while. Na przykład: 

for i in range(5):
    print(i)

while x > 0:
    print(x)
    x -= 1
Funkcje: Funkcje w Pythonie definiuje się za pomocą słowa kluczowego def. Na przykład: 

def greet(name):
    print("Hello, " + name)

greet("John")
