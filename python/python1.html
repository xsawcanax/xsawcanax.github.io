Podstawy składni Pythona

Python jest językiem programowania ogólnego przeznaczenia, który cechuje się prostotą i czytelnością składni. Oto kilka podstawowych elementów składni Pythona:

  • Zmienne: W Pythonie zmienne są deklarowane poprzez przypisanie wartości do nazwy. Na przykład: x = 10.
  • Print: Funkcja print() służy do wyświetlania danych na ekranie. Na przykład: print("Hello, World!").
  • Warunki: Warunki są wyrażane za pomocą instrukcji warunkowych if, elif, else. Na przykład: 
    
if x > 10:
    print("x jest większe niż 10")
elif x == 10:
    print("x równa się 10")
else:
    print("x jest mniejsze niż 10")
  • Pętle: Python obsługuje pętle for i while. Na przykład: 
    
for i in range(5):
    print(i)

while x > 0:
    print(x)
    x -= 1
  • Funkcje: Funkcje w Pythonie definiuje się za pomocą słowa kluczowego def. Na przykład: 
    
def greet(name):
    print("Hello, " + name)

greet("John")
